### Что реализовано на данный момент (требует описания в документации)

 - Управление активными задачами
   - Добавление новых задач с автодополнением названий.
   - Запуск/остановка таймера для каждой задачи.
   - Завершение задачи с сохранением затраченного времени.
 - Сохранённые записи
   -  Все завершённые задачи отображаются в правой панели.
   -  Возможность редактирования (имя работника, название задачи).
   -  Удаление записей.
 - Экспорт
   - Выгрузка списка сохранённых записей в Excel (формат .xlsx).
